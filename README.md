# AI Telegram Gmail GoogleSheet Automation (n8n)

## 🔹 Description

This project is an AI-powered automation workflow built using n8n. It integrates Telegram, Gmail, Google Drive, and Google Sheets to automate communication and task handling.

## 🔹 Features

* 🤖 Telegram bot to receive user messages
* 🧠 AI Agent to process user input
* 📧 Automated email sending using Gmail
* 💾 Memory support for conversation context
* 📁 Google Drive file search
* 📊 Google Sheets automation
* 🔀 Conditional logic using IF nodes

## 🔹 Tools & Technologies

* n8n
* OpenRouter (LLM API)
* Telegram API
* Gmail API
* Google Drive API
* Google Sheets API

## 🔹 Workflow

1. User sends message via Telegram
2. Workflow checks message type
3. AI processes the input
4. If email-related → Gmail action triggered
5. Otherwise → AI response generated
6. Output sent back to Telegram

## 🔹 How to Use

1. Import `workflow.json` into n8n
2. Add required API credentials
3. Activate the workflow
4. Start interacting via Telegram bot
